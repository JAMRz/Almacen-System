# Sistema de Almacén

Sistema fullstack para control de inventario de bobinas y bolsas.

## Tecnologías

- Frontend: React + Vite + Ant Design
- Backend: NestJS + PostgreSQL
- ORM: TypeORM

## Estructura

- /almacen-frontend → interfaz de usuario
- /almacen-backend → API REST

## Instalación

### Backend

cd almacen-backend
npm install
npm run start:dev

### Frontend

cd almacen-frontend
npm install
npm run dev
